<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Cognia in minutes
  </p>

  <Steps>
    <Step title="Create your account">
      Sign up for a free Cognia account to start organizing your digital knowledge.
      
      ```bash
      POST /api/v1/signup
      ```
      
      <Accordion title="Example request">
        ```json
        {
          "username": "your-username",
          "password": "your-secure-password"
        }
        ```
      </Accordion>
    </Step>

    <Step title="Authenticate">
      Sign in to get your JWT token for API access.
      
      ```bash
      POST /api/v1/signin
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "message": "Login successful",
          "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
        }
        ```
      </Accordion>
    </Step>

    <Step title="Add your first content">
      Start capturing knowledge by adding your first piece of content.
      
      ```bash
      POST /api/v1/content
      ```
      
      <Tip>You can save notes, YouTube videos, Twitter links, or any web link.</Tip>
    </Step>

    <Step title="Organize with workspaces">
      Create workspaces to organize your content by topic, project, or any way that makes sense to you.
      
      Learn more in the [Workspaces guide](/features/workspaces).
    </Step>
  </Steps>
</div>
